# AGENTS 가이드

이 문서는 이 저장소에서 동작하는 에이전트 작업 규칙을 정의한다.

## 기본 원칙
- 대상 플랫폼은 macOS 사용자 앱이다.
- 모든 답변, 문서, 코드 주석은 한글을 기본으로 작성한다.
- 앱의 기술/기능/제약사항은 `SPEC.md`를 단일 기준 문서로 확인한다.

## 문서 운영
- 변경 내역은 항상 `docs/CHANGE_LOG.md`에 기록한다.
- `docs/CHANGE_LOG.md`는 최신 항목이 상단에 오도록 유지한다.
- 변경 로그에는 날짜와 시간(로컬 타임존)을 포함한다.

## 구현/변경 규칙
- 코드나 설정을 변경하면 다음 문서 동기화를 함께 수행한다.
  - 기능/기술 정책 변경: `SPEC.md` 갱신
  - 작업 이력: `docs/CHANGE_LOG.md` 갱신
- 파일명은 `docs/CHANGE_LOG.md`를 사용한다. (`CHANEGE_LOG.md` 금지)

## 커밋/푸시 규칙
- 커밋/푸시는 기능 단위로 수행한다.
- 작업 종료 시 원격 저장소로 반영한다.
- 상세 브랜치/PR 정책은 `CONTRIBUTING.md`를 따른다.

## 보안 규칙
- API 키, 토큰, 비밀번호, 개인정보는 절대 커밋하지 않는다.
- 비밀정보는 환경변수 또는 비추적 로컬 파일로 관리한다.
- 보안 상세 정책은 `SECURITY.md`를 따른다.
